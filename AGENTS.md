# Instructions for coding agents

This repository is a writing skill. Before writing any prose a person will read (PR descriptions, commit messages, docs, comments, chat replies, social posts), read `SKILL.md` and follow it while drafting. Run its self-audit section before delivering the text.

`SKILL.md` is plain markdown with no tool or platform dependencies. The grep command near the end is optional; use it if you have shell access, otherwise apply the checks by reading.
